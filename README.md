插件功能
解析各大视频网站视频，支持优酷，爱奇艺，乐视，腾讯视频，土豆，芒果TV，bilibili，PPTV，暴风，音悦台，适合某些网络下无法打开视频网站的情况。
使用必读
安装插件后在视频播放页面的左侧会显示解析按钮，点击后选择解析线路点击后即可播放
